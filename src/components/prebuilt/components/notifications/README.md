# Notifications Components

Production-ready notification components with animations, accessibility, and responsive design.

## Components

| Component | Description | File |
|-----------|-------------|------|
| ToastSystem | Stackable toast notifications with auto-dismiss | [ToastSystem.md](./ToastSystem.md) |
| NotificationCenter | Bell icon dropdown with notification list | [NotificationCenter.md](./NotificationCenter.md) |
| AlertBanner | Full-width alert banners | [AlertBanner.md](./AlertBanner.md) |
| SnackBar | Bottom-positioned snackbar with actions | [SnackBar.md](./SnackBar.md) |

## Usage

Each component is documented in its own `.md` file with:
- Full TypeScript implementation
- Props interface
- Usage examples
- Accessibility notes

## Features

All notification components include:
- ✅ TypeScript types
- ✅ Smooth animations
- ✅ Auto-dismiss timers
- ✅ Pause on hover
- ✅ Keyboard navigation
- ✅ Screen reader announcements (aria-live)
- ✅ Focus management
- ✅ Memory leak prevention (timeout cleanup)
